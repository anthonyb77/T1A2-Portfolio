# T1A2 Portfolio - Anthony Beck

- Site URL: https://anthonybeck.dev
- Code Repository: https://github.com/anthonyb77/t1a2-portfolio

## Purpose

This is my personal online portfolio. Made to provide information about myself, my professional knowledge and skills, and showcase my work to potential employers. It also served as a learning experience in website design, development, and deployment.

## Functionality / Features

- Mobile first design with responsive scaling to larger screens.
- Using Flexbox and CSS Grid.
- Accessible design.
- CSS Animations.
- Showcase of my skills and projects.
- Personal information.
- Contact information.

## Target Audience

This portfolio is intended to be viewed by employers seeking a software developer or IT professional.

## Tech Stack

- HTML5
- CSS
- SASS
- NPM
- Javascript
- Visual Studio Code (IDE), with *Live Server* extension.
- Unsplash (some images)
    - Free to use images https://unsplash.com/license
- Google Fonts (fonts)
    - Google Font repository https://fonts.google.com/
- Figma (wireframing)
    - https://figma.com/
- Apache 2 (web server)
- DigitalOcean (deployment platform)
    - I already had a server with Digital Ocean for personal projects and decided to host it here, I uploaded the .html to the web server directory for my domain and set appropriate configuration settings to make it available publicly.
- Developed on Windows / Ubuntu (WSL)
